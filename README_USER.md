# 📘 个人品牌网站操作手册 (README_USER)

欢迎使用您的个人品牌网站管理系统！本手册将指导您如何通过简单的操作来更新全站内容、管理留言和发布新文章。

---

## 🚀 1. 如何进入后台管理

后台管理系统是您控制网站内容的“驾驶舱”。

1.  **访问地址**: 在浏览器地址栏输入 `http://您的域名/admin`（本地开发时通常是 `http://localhost:3000/admin`）。
2.  **导航菜单**: 进入后，左侧边栏列出了所有可管理的功能模块：
    *   **Dashboard**: 概览（目前主要作为入口）。
    *   **首页管理**: 修改 Slogan、封面图、社交链接。
    *   **关于我**: 编辑个人简介、能力标签、更新时间轴履历。
    *   **项目进展**: 更新核心业务介绍和未来规划。
    *   **分享与手记**: **发布新文章的核心区域**。
    *   **留言管理**: 查看所有访客的留言。
    *   **全局设置**: 修改网站标题、Logo、测试邮件发送。

---

## ✍️ 2. 如何发布一篇新的“创业分享”文章

无需写代码，只需在后台操作即可：

1.  点击左侧菜单的 **“分享与手记”**。
2.  在顶部 Tab 切换到 **“创业分享 (Entrepreneurship)”**。
3.  点击列表右上角的 **“+ 添加文章”** 按钮，列表中会出现一个新的空白卡片。
4.  填写内容：
    *   **标题**: 文章的主标题。
    *   **日期**: 发布日期（默认当天）。
    *   **阅读量**: 手动输入初始阅读量（如 0）。
    *   **分类**: 在下拉框中选择分类（如“实战复盘”）。如果需要新分类，请先在顶部的“板块设置”中添加。
    *   **摘要**: 简短介绍文章核心内容。
    *   **封面图 URL**: 粘贴图片的链接（推荐使用 Unsplash 或您自己的图床链接）。
    *   **置顶 (Sticky)**: 勾选后，该文章将强制显示在列表最前面，并带有“置顶”标签。
5.  点击页面右上角的 **“保存更改”** 按钮。
    *   *注意：必须点击保存，修改才会写入文件并生效！*

## 🛠️ 3. 如何通过修改 siteData.json 快速更新全站

如果您更习惯直接编辑文件，或者需要进行批量修改，可以使用“上帝模式”：

1.  在项目文件夹中找到文件：`web/src/data/siteData.json`。
2.  这是一个标准的 JSON 格式文件，包含了全站所有文字信息。
3.  **修改规则**：
    *   找到对应的字段直接修改文字（保留双引号）。
    *   例如修改首页 Slogan：
        ```json
        "home": {
          "slogan": "这里写您的新标语",
          ...
        }
        ```
4.  **保存文件** (`Ctrl + S`)。
5.  网站前台会**立即自动更新**，无需重启服务。

## 📧 4. 邮件与留言系统

*   **查看留言**: 后台点击 **“留言管理”**，可以看到所有用户提交的表单信息。
*   **邮件通知**: 
    *   当有新留言时，系统会尝试发送邮件到您设置的 `contactEmail`。
    *   **测试邮件**: 进入 **“全局设置”**，点击 **“发送测试邮件”** 按钮，验证邮件服务是否配置正确。
    *   *注意：发送真实邮件需要配置环境变量 `EMAIL_USER` 和 `EMAIL_PASS`。*
